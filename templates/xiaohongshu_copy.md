# 小红书爆款种草笔记文案框架

## 适用场景
- 商品种草笔记（穿搭/家居/好物分享）
- 图文笔记为主，视频笔记结构类似
- 目标：收藏+评论+转化

## 结构模板（8段式）

### 1. 标题（15-25字）
- 情绪化+悬念/数字/对比
- 高分示例："被问爆了！这套睡衣真的舒服到不想脱"
- 低分示例："男士睡衣推荐"

### 2. 开场（2-3句）
- 个人场景代入，有具体时间/地点/人物
- 高分示例："最近宅家时间变多了，之前那套旧睡衣洗多了袖口都磨毛了"
- 低分示例："今天给大家介绍一款睡衣"

### 3. 痛点（3-4句）
- 描述之前的糟糕体验，引发共鸣
- 用第一人称"我"
- 可以带一点吐槽感

### 4. 发现（2-3句）
- 怎么遇到这个产品的
- 有故事性：闺蜜安利/刷到/逛街偶然看到
- 不要直接说"品牌方给我寄了样品"

### 5. 体验（5-8句，核心段落）
- 具体使用感受，多感官描述
- 触感：rua一下/贴皮肤/冰冰凉凉
- 温度：夏天不闷汗/冬天不冰凉
- 版型：宽松但不垮/显瘦/不勒腰
- 至少1个意外发现的细节

### 6. 细节（2-3句）
- 1-3个打动人的小设计
- 如：滚边不磨皮肤/口袋够深能装手机/纽扣不会掉

### 7. 总结（2句）
- 一句话推荐+适用人群
- 可以带价格锚定（"不到一杯奶茶钱"）

### 8. 互动（1句）
- 提问引导评论
- 如："你们宅家都穿什么？""有没有同款举手"

## 去 AI 感核心约束

### 必须避免的结构
- [ ] "首先...其次...最后..." 的并列结构
- [ ] "众所周知""不得不说""值得一提的是"
- [ ] "让我们来了解一下""接下来我要说的是"
- [ ] 完美对称的排比句
- [ ] 每段开头格式一致（如都以"我"开头）

### 必须保留的不完美
- [ ] 句子长短不一，有短到1个词或长到30+字的句子
- [ ] 有错别字或不完整短句（如"就...挺意外的"）
- [ ] 口语化连接词（"然后吧""反正""其实""那个"）
- [ ] 个人口头禅或地域特色（"咱就是说""尊嘟假嘟""绝绝子"等，适度）
- [ ] emoji 1-3个，位置随意，不在句尾对齐
- [ ] 有明确的时间/地点/人物（"上周六在闺蜜家""我对象说..."）

## 短文案（主图slogan）
- 1-2句，情绪化表达
- 高分："像把云穿上身""舒服到不想出门"
- 低分："高品质面料，舒适透气"
